# 怎么学习
遵循wwh原则：
+ What:要学的东西（技术）是什么？
可以从官方文档的定义，这个技术的架构组成，还有对比这个东西（技术）与同类的优缺点，适用场景，使用方式等。

+ Why:为什么会产生这个东西（技术）？
这个技术解决了什么问题，如果直接解决这个问题要怎么解决，还有没又其他方式解决。

+ How：怎么学习这个东西（技术）？
技术从哪里来，在哪里学最快，所以官网是学习的好地方，还有就是相关的书籍，技术文档,写demo，看源码，在项目中进行实践，总结分享等。
