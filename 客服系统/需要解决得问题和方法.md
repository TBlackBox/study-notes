# 怎么快速得实现数据备份？
1. 在客服总后台下提供一个页面，可以下载数据库sql文件

# 怎样进入总后台
1. 运行起项目后，进入总后台地址，第一次需要注册超级管理员账号，如果有备份数据库，可以上传数据库文件，这样对数据得迁移非常得方便。
2.  创建完管理员账号后，后台回根据你的账号生产一个token（密钥）,这个token将是你添加客服人员需要得。必须妥善保管。

# 怎样添加客服
1. 通过下面的接口添加客服
```
HTTP://域名/接口地址
```

# 怎样实现数据创建表或将sql文件转换为数据
1. 使用jpa，后面可能需要使用好都datebase.
