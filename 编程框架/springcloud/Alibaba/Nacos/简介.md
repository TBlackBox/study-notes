# 简介

Nocos 是由Nameing和Configuration的前两个字母，最后的s是service.



一个更易于构建云原生应用的动态服务发现，配置管理和服务管理中心,Nacos也就是注册中心+配置中心的组合，等价于

```
Nacos = Eureka+Config+Bus
```
