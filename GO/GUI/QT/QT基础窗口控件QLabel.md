# 简介

QLabel可以显示不可编辑的文本或图片，也可以被用作提示标记为其他控件，继承自QFrame。
常用的方法

1. setAlignment()：设置文本的对齐方式

2. setIndent()：设置文本缩进

3. text()：获取文本内容

4. setBuddy()：设置伙伴关系

5. setText()：设置文本内容

6. selectedText()：返回所选择的字符

7. setWordWrap()：设置是否允许换行

   

## 常用的信号(事件)

- 当鼠标划过QLabel控件时触发：linkHovered
- 当鼠标单击QLabel控件时出发：linkActivated
