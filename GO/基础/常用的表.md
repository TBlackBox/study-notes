Go 语言的字符串常见转义符包含回车、换行、单双引号、制表符等，如下表所示。

| 转义 | 含义                               |
| ---- | ---------------------------------- |
| \r   | 回车符（返回行首）                 |
| \n   | 换行符（直接跳到下一行的同列位置） |
| \t   | 制表符                             |
| \'   | 单引号                             |
| \"   | 双引号                             |
| \    | 反斜杠                             |


## 输出相关的符号
```

%d 十进制整数 

%x, %o, %b 十六进制，八进制，二进制整数。 

%f, %g, %e 浮点数： 3.141593 3.141592653589793 3.141593e+00 

%t 布尔：true或false 

%c 字符（rune） (Unicode码点) 

%s 字符串 

%q 带双引号的字符串"abc"或带单引号的字符'c' 

%v 变量的自然形式（natural format） 

%T 变量的类型 

%% 字面上的百分号标志（无操作数） 
```
